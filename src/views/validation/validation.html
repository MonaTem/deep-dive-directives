Built-in Validation

Every time we use the form directive, Angular instantiates the FormController and publish it on the current scope (we need to give our form a name for reference).

A simple form for example:

{{loginForm.userName | json}}

Custom Validation

by requiring the form directive in our custom directive, we can gain access to this controller methods and properties.

You did not enter a field
Must be at least 2 chars!
Name already exist!
{{ customForm.nameInput.$error | json }}
{{ customForm.userInput.$error | json }}
